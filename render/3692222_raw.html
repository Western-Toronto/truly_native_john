Make Batch Files

A tool that allows to create Windows batch files quickly. Eeasy to learn and to use.

  • No need to remember or type command names. Just drag-and-drop items with understandable names to the project.
  • No need to learn or remember all of the switches. Visual options editor for every command!
  • Automates common tasks like validating parameters or showing "how to use" info.
