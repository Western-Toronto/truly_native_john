PhoneGap Tutorial: Building a Camera App with PhoneGap and Visual Studio 2015

  • By Tom Faltesek +
  • March 19, 2015
  • Apache Cordova CSS HTML Hybrid App JavaScript PhoneGap Windows Phone
  • 2 Comments
  • Loading...

A few years ago I spent some time exploring PhoneGap (Apache Cordova), but my interest was diverted towards mobile and single page web applications. However, the recent release of the Visual Studio Tools for Apache Cordova caught my attention. Front-end web developers can easily leverage their powers to create cross-platform, hybrid mobile applications without much spin up time. The new Visual Studio integration with Apache Cordova lowers the barrier to entry that much more for Visual Studio users. We no longer have to worry about setting up a hodgepodge of tools and emulators. With Visual Studio 2015 (and 2013 Update 4) we can jump right in and have an application deployed to our devices in minutes.

If youre a front-end web developer whos a bit late to the hybrid mobile app game like me, dont fret. Theres still time for you to give it a shot and its easier than ever now to get started. I put together a quick PhoneGap tutorialvideo to provide you with evidence and perhaps some inspiration.

The PhoneGap tutorial video willcover:

  • Visual Studio/Apache Cordova Prerequisites
  • Starting a new Apache Cordova project
  • Integrating a common PhoneGap native plugin: The Camera
  • Deploying to a Windows Phone 8.1

There you have it. In less than fifteenminutes we managed to create and deploy a hybrid app that uses device-specific hardware (the camera). Its not pretty or complex, but the point is that three years ago we (definitely me!) may have spent a Saturday to achieve the same results.

Im hoping that Ive left you pumped up and ready to make some apps. The URLs I referenced in the video can be found below. Happy coding!
