Top 5 Software Quality eBooks of 2013 (Free for Download)

December 17, 2013 by Ben Austin Leave a Comment

This is the time of the year that companies feel inclined to send out special holiday deals andseasonal greetings meant to make you feel all warm inside. As nice as that may be, these holiday messages dont usually provide you with anything of value.

Rather than spend time trying to show off our holiday cheer we have, we decided it made more sense to just give out five of our top eBooks from 2013 for free. As in, completely free.No cost.No email address.No name. No gimmick. We truly just want to allow you to download these eBooks to thank you for visiting our blog, and the only reason you would need to provide us with any information is if you decided to download one of our complimentary trials.

Whether youre a functional tester, a sysadmin, a developer, or some far-out fusion of the three, I hope youre able to find some valuable information in the resources listed below.

A Step-by-Step Guide to Cross-Browser Testing

Do you test your web applications in multiple browsers? If you do, you know that automation of tests for different browsers can be annoyingly difficult, especially with modern Web 2.0 applications.

Modern test automation tools provide built-in support for cross-browser testing that allows tests created for one browser to be played back in other supported browsers including IE 10, Mozilla Firefox and Google Chrome.. Additionally, distributed testing capabilities allow you to run cross-browser tests against different browsers on multiple machines at the same time. Share this.

The State of Code Review

We launched an extensive study about code review practices, software quality, and how the two are related (or not!). We attracted respondents to our survey by placing ads in various developer-oriented online properties, as well as contacting practitioners who are on our mailing list.

Included in this report, you will receive code review and software quality performance benchmarks for participants in your industry, your organization size, and your development practices. You will learn what the most successful organizations are doing differently to get ahead, and which best practices you can apply for your own improvements in software quality. Share this.

SoapUI 101: Beginners Guide to Functional Testing

While SoapUIs functionality includes functional testing and security testing, as well as service simulation, functional testing is definitely the most popular.

Functional testing means checking that the behavior of the system under consideration matches the business expectations in terms of functionality. SoapUI enables users to perform functional testing easily. Share this.

A Step-by-Step Guide to Functional Testing with TestComplete

Not all test automation tools are created equal, especially in the case of functional testing. The best choice is an automation tool that allows you to not only create automated tests, but also fully automate the whole QA testing process.

In this case, the automation tool becomes the QA testers assistant. It can perform QA testing actions, such as running scheduled automated tests, checking results, sending reports to different issue-tracking systems and a lot more. Grab this new SmartBear eBook and learn how TestComplete can be a game-changer for your team. Share this.

Load Testing 101: Essential Tips for Testers and Developers

If you are interested in learning the basics of software load testing,weve put together an eBookthat will help.

In this eBook, youll learn how to:

  • Prepare for optimal load testing of a web application
  • Record scenarios
  • Test in a way that emulates real life conditions
  • Analyze your load testing data for the highest quality software

Share this.
